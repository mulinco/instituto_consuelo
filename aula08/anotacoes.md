## DOM - Seleção e Modificação de Eventos  
- map(): tranforma um array original num novo
- filter(): é um método de arrays que cria um novo array com todos os elementos que passam em um teste especificado por uma função de retorno de chamada (callback). Ele não modifica o array original e funciona passando uma função que deve retornar true ou false para cada item; apenas os que retornam true são incluídos no novo array.
- find(): O método find() em JavaScript retorna o primeiro elemento de um array que satisfaz uma condição especificada em uma função de callback. Se nenhum elemento atender à condição, ele retorna undefined. 
- reduce():  é uma função de ordem superior que executa uma função reducer para cada elemento em um array, resultando em um único valor. Ele é usado para "reduzir" um array a um único valor, como soma, média, ou encontrar o maior/menor elemento, por exemplo.
